# Clover-X79-E5-2630LV2-GTX650
Hackintosh Clover for X79

EFI适用于X79安装黑苹果

## 主机配置：

- CPU：E5 2630L V2 2.4G
- 主板：国硕X79（Q65）寨板
- 内存：ECC DDR3 1600 8G *2 
- GPU：蓝宝石 RX460 白金版4G（刷RX560）
- 蓝牙：奥睿科USB

## 系统支持：

	- 10.13.X
	- 10.14.X
	- 10.15.X

## 已测试：

- 10.13.6：各项功能正常
- 10.14.2：可用，睡眠好像不太好，台式机不用睡眠也无所谓；
- 10.14.6：同上
- 10.15.3：同上
